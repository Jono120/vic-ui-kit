**Requirements**: - `id` of the element with class 'popup' has to be unique.

**Function openPopup() parameters**: - `optionsObject` (optional): Valid JSON Object allowing to customise auto-loading behaviour. Properties: `delayInMs` (default: 0) - How long after the page loads should the popup box appear. `suppressAfterCanceling` (default: false) - Once the user cancels or closes the popup, the popup will (not) load again next time the same page is loaded. --- Open popup box on click
