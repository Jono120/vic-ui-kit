--- notes: > Popup box that opens automatically after a page is loaded.

**Requirements**: - `id` of the element with class 'popup' has to be unique.

**HTML attributes**: - `data-autoload` (required): Initialises the auto-loading. - `data-opts` (optional): Valid JSON Object allowing to customise auto-loading behaviour. Properties: `delayInMs` (default: 3000) - How long after the page loads should the popup box appear. `suppressAfterCanceling` (default: true) - Once the user cancels or closes the popup, the popup will (not) load again next time the same page is loaded. ---
