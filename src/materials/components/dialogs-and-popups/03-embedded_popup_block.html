--- notes: > Popup block embedded in a conten. Can be opened programatically, e.g. on click to an element.

**Requirements**: - `id` of the element with class 'popup' has to be unique. - element with class 'popup' is not wrapped in `div.popup-positioner`

**Function openPopup() parameters**: - `optionsObject` (optional): Valid JSON Object allowing to customise auto-loading behaviour. Properties: `delayInMs` (default: 0) - How long after the page loads should the popup box appear. `suppressAfterCanceling` (default: false) - Once the user cancels or closes the popup, the popup will (not) load again next time the same page is loaded. --- Open embedded popup block on a click
